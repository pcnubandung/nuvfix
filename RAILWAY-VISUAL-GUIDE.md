## Next Steps

### 1. Ganti Password Admin
- Menu: Pengaturan > Manajemen User
- Edit user admin
- Set password baru yang kuat

### 2. Setup Data Koperasi
- Menu: Informasi Koperasi
- Isi nama, alamat, kontak
- Upload logo

### 3. Tambah Anggota
- Menu: Anggota > Tambah Anggota
- Atau gunakan form pendaftaran publik

### 4. Custom Domain (Optional)
- Settings > Domains > Add Custom Domain
- Update DNS CNAME record

### 5. Backup Database
```bash
railway run cat /app/data/koperasi.db > backup.db
```
