1. PENDAHULUAN

1.1 Tentang Koperasi NU Vibes

Koperasi NU Vibes adalah sistem manajemen koperasi berbasis web yang dirancang untuk membantu pengelolaan koperasi secara modern, efisien, dan terintegrasi. Aplikasi ini menyediakan fitur lengkap mulai dari manajemen anggota, transaksi simpanan, keuangan, hingga perhitungan Sisa Hasil Usaha (SHU).

1.2 Fitur Utama

  • Dashboard Interaktif - Statistik real-time dengan grafik Chart.js
  • Manajemen Anggota - CRUD lengkap dengan foto dan import/export
  • Transaksi Simpanan - 4 jenis simpanan dengan cetak struk
  • Keuangan Terintegrasi - Penjualan, pengeluaran, dan laporan
  • Perhitungan SHU Otomatis - Berdasarkan simpanan dan partisipasi
  • Multi-level User - Admin, Pengurus, dan Kasir
  • Responsive Design - Bisa diakses dari PC, tablet, dan smartphone

1.3 Teknologi yang Digunakan

Komponen Teknologi Versi
Backend Node.js + Express.js v18+
Database SQLite3 v5.1+
Frontend HTML5, CSS3, JavaScript ES6+
Charts Chart.js v4.4+
Icons Feather Icons Latest

2. INSTALASI & SETUP

2.1 Persyaratan Sistem

Komponen Minimum Rekomendasi
Operating System Windows 10, macOS 10.14, Ubuntu 18.04 Windows 11, macOS 12+, Ubuntu 20.04+
RAM 4 GB 8 GB
Storage 1 GB 5 GB
Browser Chrome 90+, Firefox 88+ Chrome Latest, Firefox Latest

2.2 Instalasi Node.js

1Download Node.js

  • Kunjungi https://nodejs.org/
  • Download versi LTS (Long Term Support)
  • Pilih sesuai sistem operasi Anda

2Install Node.js

  • Jalankan file installer yang sudah didownload
  • Ikuti wizard instalasi dengan pengaturan default
  • Pastikan opsi "Add to PATH" dicentang

3Verifikasi Instalasi

node --version
npm --version

2.3 Setup Aplikasi

1Extract File Aplikasi

  • Extract file aplikasi ke folder yang mudah diakses
  • Contoh: C:\koperasi-nu-vibes\

2Install Dependencies

cd koperasi-nu-vibes
npm install

3Jalankan Aplikasi

npm start

# Atau double-click file start.bat (Windows)

4Akses Aplikasi

  • Buka browser
  • Akses: http://localhost:3000/login.html
Berhasil! Jika muncul halaman login, instalasi sudah berhasil.

11. SISA HASIL USAHA (SHU)

11.1 Tentang SHU

Sisa Hasil Usaha (SHU) adalah keuntungan koperasi yang dibagikan kepada anggota berdasarkan kontribusi mereka. Perhitungan SHU mengikuti prinsip koperasi dan peraturan yang berlaku.

11.2 Komponen Perhitungan SHU

Komponen Persentase Deskripsi
Jasa Modal 40% Berdasarkan total simpanan anggota
Jasa Usaha 40% Berdasarkan partisipasi transaksi
Dana Cadangan 10% Untuk cadangan koperasi
Dana Pengurus 5% Untuk pengurus koperasi
Dana Pendidikan 5% Untuk pendidikan anggota

11.3 Cara Hitung SHU

1Buka menu "Sisa Hasil Usaha"

2Klik "Hitung SHU Baru"

3Pilih periode (tahun)

4Masukkan total SHU yang akan dibagikan

5Sistem akan otomatis menghitung:

  • Total simpanan per anggota
  • Total partisipasi per anggota
  • Jasa modal per anggota
  • Jasa usaha per anggota
  • Total SHU per anggota

6Review hasil perhitungan

7Klik "Simpan" untuk menyimpan hasil

8Export ke PDF untuk dokumentasi

11.4 Rumus Perhitungan

SHU Anggota = Jasa Modal + Jasa Usaha

Jasa Modal = (Simpanan Anggota / Total Simpanan) × 40% × Total SHU

Jasa Usaha = (Partisipasi Anggota / Total Partisipasi) × 40% × Total SHU
Otomatis & Akurat: Sistem menghitung SHU secara otomatis berdasarkan data simpanan dan partisipasi yang tercatat sepanjang tahun.

13. TROUBLESHOOTING

13.1 Masalah Umum & Solusi

Problem: Aplikasi tidak bisa dijalankan

Solusi:
  • Pastikan Node.js sudah terinstall (cek dengan: node --version)
  • Jalankan npm install untuk install dependencies
  • Cek apakah port 3000 sudah digunakan aplikasi lain
  • Restart komputer dan coba lagi

Problem: Tidak bisa login

Solusi:
  • Pastikan username dan password benar (admin/admin123)
  • Cek apakah Caps Lock aktif
  • Clear browser cache dan cookies
  • Coba browser lain (Chrome/Firefox)
  • Reset database jika perlu

Problem: Menu tidak muncul / stuck "Memuat data..."

Solusi:
  • Buka Console Browser (F12) dan cek error
  • Refresh halaman (Ctrl + Shift + R)
  • Logout dan login kembali
  • Cek koneksi ke server (http://localhost:3000)
  • Restart aplikasi

Problem: Error saat upload foto

Solusi:
  • Pastikan ukuran file tidak lebih dari 5MB
  • Gunakan format JPG atau PNG
  • Cek folder uploads/ ada dan writable
  • Compress foto jika terlalu besar

Problem: Grafik tidak muncul

Solusi:
  • Pastikan ada data transaksi
  • Refresh halaman
  • Clear browser cache
  • Cek Console Browser untuk error Chart.js

Problem: Export PDF/Excel tidak berfungsi

Solusi:
  • Pastikan browser mengizinkan download
  • Cek popup blocker tidak aktif
  • Coba browser lain
  • Pastikan ada data untuk di-export

13.2 Error Database

Error: "SQLITE_ERROR: no such table"

Penyebab: Tabel database belum dibuat atau corrupt
Solusi: Reset database dengan npm run reset-db

Error: "database is locked"

Penyebab: Database sedang diakses oleh proses lain
Solusi:
  • Stop semua instance aplikasi
  • Tutup aplikasi yang mengakses file .db
  • Restart aplikasi

13.3 Debugging

Untuk debugging lebih lanjut:

Browser Console (F12):

  • Tab Console - Lihat error JavaScript
  • Tab Network - Cek API calls dan response
  • Tab Application - Cek localStorage dan session

Server Console:

  • Lihat terminal tempat aplikasi berjalan
  • Cek error message dan stack trace
  • Perhatikan status code HTTP (200, 400, 500, dll)
Tips: Screenshot error message dan console log untuk memudahkan troubleshooting.

14. TIPS & TRIK

14.1 Best Practices

Setup Awal:

  1. Ganti password default segera setelah login pertama
  2. Isi informasi koperasi lengkap dengan logo
  3. Buat unit usaha sebelum input transaksi
  4. Tambah anggota dengan data lengkap

  1. Input transaksi simpanan setiap hari
  2. Catat penjualan dan pengeluaran secara rutin
  3. Cetak struk untuk setiap transaksi
  4. Cek dashboard untuk monitoring real-time
  5. Backup data secara berkala

Akhir Periode:

  1. Generate laporan bulanan
  2. Review laba/rugi per unit usaha
  3. Export laporan ke PDF untuk arsip
  4. Hitung SHU di akhir tahun
  5. Backup database sebelum tahun baru

14.3 Tips Keamanan

  • Password Kuat - Minimal 8 karakter, kombinasi huruf, angka, simbol
  • Ganti Password Berkala - Setiap 3-6 bulan
  • Jangan Share Akun - Buat user terpisah untuk setiap orang
  • Logout Setelah Selesai - Terutama di komputer bersama
  • Backup Rutin - Minimal seminggu sekali
  • Update Aplikasi - Jika ada versi baru

14.4 Tips Backup & Restore

Backup Manual:

1Stop aplikasi

2Copy file koperasi.db ke folder backup

3Beri nama dengan tanggal (contoh: koperasi-2024-11-08.db)

4Simpan di lokasi aman (external drive, cloud storage)

Restore dari Backup:

1Stop aplikasi

2Hapus atau rename file koperasi.db yang lama

3Copy file backup ke folder aplikasi

4Rename menjadi koperasi.db

5Jalankan aplikasi kembali

14.5 Tips Performa

  • Gunakan Browser Modern - Chrome atau Firefox versi terbaru
  • Clear Cache Berkala - Untuk performa optimal
  • Tutup Tab Tidak Terpakai - Hemat memory
  • Restart Aplikasi - Jika terasa lambat
  • Compress Foto - Sebelum upload untuk hemat storage

14.6 Tips Export & Print

  • Export Excel - Untuk analisis dan edit data
  • Export PDF - Untuk arsip dan dokumentasi resmi
