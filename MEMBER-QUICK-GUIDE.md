## â“ FAQ

### Q: Kenapa data saya kosong?
**A:** Kemungkinan Anda belum punya transaksi simpanan atau partisipasi. Hubungi admin untuk menambahkan data.

### Q: Bagaimana cara menambah simpanan?
**A:** Simpanan ditambahkan oleh admin/kasir koperasi. Anda bisa setor ke kasir, lalu mereka akan input ke sistem.

### Q: Estimasi SHU saya akurat?
**A:** Ini hanya estimasi berdasarkan perhitungan sederhana. SHU aktual akan diumumkan di RAT dan bisa berbeda.

### Q: Bisa download laporan?
**A:** Fitur download belum tersedia. Untuk sementara bisa screenshot atau minta laporan ke admin.

### Q: Lupa password, bagaimana?
**A:** Hubungi admin koperasi untuk reset password Anda.

### Q: Bisa lihat data tahun lalu?
**A:** Saat ini menampilkan semua data historis. Filter per tahun belum tersedia.

### Q: Kenapa tidak bisa edit simpanan?
**A:** Member hanya bisa melihat data. Edit/tambah simpanan hanya bisa dilakukan admin/kasir.

## ğŸ†˜ Troubleshooting

### Tidak bisa login
1. Pastikan username/email benar
2. Pastikan password benar (default: `member123`)
3. Cek status anggota (harus aktif)
4. Clear browser cache
5. Hubungi admin jika masih gagal

### Dashboard loading terus
1. Refresh halaman (F5)
2. Clear localStorage: `localStorage.clear()` di Console
3. Logout dan login ulang
4. Cek koneksi internet
5. Hubungi admin jika masih bermasalah

### Menu tidak bisa diklik
1. Refresh halaman (Ctrl+F5)
2. Cek Console untuk error (F12)
3. Clear cache browser
4. Coba browser lain

### Chart tidak muncul
1. Pastikan ada data simpanan
2. Refresh halaman
3. Cek Console untuk error
4. Pastikan Chart.js loaded
