CARA MENJALANKAN
----------------
1. Install Node.js dari https://nodejs.org/
2. Buka Command Prompt di folder aplikasi
3. Jalankan: npm install
4. Jalankan: npm start
5. Buka browser: http://localhost:3000/login.html
6. Login dengan: admin / admin123

ATAU (Windows):
1. Double-click file start.bat
2. Tunggu hingga server berjalan
3. Buka browser: http://localhost:3000/login.html

TROUBLESHOOTING
---------------
Problem: npm tidak dikenali
Solution: Install Node.js dari https://nodejs.org/

Problem: Port 3000 sudah digunakan
Solution: Ubah PORT di server.js atau stop aplikasi lain

Problem: Database error
Solution: Hapus file koperasi.db dan restart server

Problem: Upload error
Solution: Buat folder 'uploads' di root project

Problem: Tidak bisa login
Solution: Gunakan admin/admin123 atau reset database
